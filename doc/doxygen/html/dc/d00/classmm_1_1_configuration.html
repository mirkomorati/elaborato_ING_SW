Descrizione dettagliata

Singleton che mantiene le configurazioni dell'intera applicazione.

I parametri di configurazione dell'applicazione devono essere salvati in un file in formato json. La classe si occupa sia di leggere che di fornire un'interfaccia semplice per ottenere un parametro dal file
